Mini Aplicação para Controle de Funcionários

Este pequeno app foi feito como exercício (individual) da disciplina Programação para Dispositivos Móveis. A aplicação permite adicionar, deletar e editar os dados de funcionários e cargos de uma empresa. Foi utilizado o SQLite para persistir os dados gerados na aplicação.
