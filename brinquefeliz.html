Aplicação Web BrinqueFeliz (Front)

Este projeto foi o trabalho final da disciplina Análise e Projeto de Aplicações Web. O trabalho foi realizado em grupo e eu contribui com a parte de front-end e documentação. Também fui responsável pelo design do logo da aplicação para o nosso grupo. O professor passou um documento com a especificação de requisitos da aplicação e então, em grupos, os alunos começaram o processo de desenvolvimento do software. O documento era sobre um sistema de gerenciamento de vendas de brinquedos.

HTML e CSS

Foi com a implementação das telas dessa aplicação que tive, pela primeira vez, um maior contato com as linguagens HTML e CSS. Pude ter uma maior noção de como funcionava a dinâmica dessas linguagens.

As duas telas são apenas interfaces gráficas, formadas apenas por HTML e CSS. Clique aqui para navegar por essas páginas.
Obs: foram projetadas para Desktop, apenas.
